## ğŸ”§ Customization

### **Update Content**

Edit text directly in `index.html` within respective section tags.

### **Change Colors**

Modify color variables in `style.css`:

```css
background: #00629b; /* Primary color */
color: #1a1a1a; /* Text color */
```

### **Replace Images**

Add new images to `images/` folder and update `src` attributes in HTML.

### **Adjust Animations**

Modify animation duration in CSS:

```css
animation: slide 12s infinite; /* Hero slider */
animation: scroll-logos 25s linear; /* Logo marquee */
```
